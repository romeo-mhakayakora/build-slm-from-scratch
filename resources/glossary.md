# Glossary

- **SLM**: Small Language Model.
- **BPE**: Byte Pair Encoding (a common subword tokenization algorithm).
- **Tokenizer**: Converts text into token IDs (and back).
- **Vocabulary (vocab)**: Mapping from tokens to integer IDs.
- **Context length**: Max tokens the model attends to.
- **Checkpoint**: Saved model state during/after training.
- **Inference**: Generating text from a trained model.
